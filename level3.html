SWEEET! now lets write our own functions

How you ask?

  • like const or let vars we need to use a keyword to declare a function
  • keyword is: function
  • below is the code for declaring a function hello
      
        function hello(){
          ...make it do things..
        }

Lets make this function print out a message in the console

  • open up level3.js in vscode
  • on line 6 replace "...make it do things..." with your code
  • then to "call" or activate your function write the function's name followed by "()"
        
          function hello(){
            console.log("this message!!!!");
          }
          hello();

Now lets add arguments to this function to customize our message that is being printed

  • open up level3.js in vscode
  • on line 5 add an argument of "message", containing your message, to your function hello
  • remember to also pass in your argument to your function call
  • then open up your web console and check that your message is being printed
        
          function hello(message){
            console.log(message);
          }
          hello("this is a cool message!");

Finally lets make a function that will add two numbers for us and actually return a value

  • open up level3.js in vscode
  • uncomment lines 13-15
  • we want to be able to input 2 values in to the function and recieve back 1 value (the addition of the 2 previous values)
  • our inputs are our arguments, lets call them "a" and "b"
  • where in our function do we place "a" and "b"
        
          because "a" and "b" are our inputs to the function
          they will go into the function as arguments, seperated by commas
          
          function add(a,b){
             // doing something...
          }
  • we want function add to return our added two values
  • we can add "a" and "b" with the addition operator "+"
  • and we can tell our function to return a value with the keyword "return"
  • add the two arguments together and return that value
        
          function add(a,b){
             return a+b;
          }
  • we can "call" our function and store its return value in a variable
  • let's do that and console.log or document.write our variable to see our function ADD
        
          function add(a,b){
             return a+b;
          }

          const added = add(2,3);


          // you can pass as many arguments as you would like to print
          // in your console.log
          // make sure to wrap strings with quotes
          // and make sure to seperate arguments with commas
          console.log("2 + 3 = ", added);
