An array is a list of items contained within brakets that looks like this 
      
        let anArray = ["one", 2, 3];

Arrays hold information about what position a value is in the list, so to access them you can use their index values that correspond to their order.

For example to get the first value from "anArray", we would do the following 
      
        let anArray = ["one", 2, 3];
        let firstValue = anArray[0] --> will get you value "one";

to keep getting more values further down we would increase our index value

other info you can get from an array is its length 
      
        let anArray = ["one", 2, 3];
        let length = anArray.lenght() --> will get you value 3;
