About Arrays

An array is a list of items contained within brakets that looks like this 
      
        let anArray = ["one", 2, 3];

Arrays hold information about what position a value is in the list, so to access them you can use their index values that correspond to their order.

For example to get the first value from "anArray", we would do the following 
      
        let anArray = ["one", 2, 3];
        let firstValue = anArray[0]; --> will get you value "one"

to keep getting more values further down we would increase our index value

other info you can get from an array is its length 
      
        let anArray = ["one", 2, 3];
        let length = anArray.length(); --> will get you value 3

First lets breakdown the loop

To loop through an array we can use the for loop syntax

    
      for(let i = 0; i < items.length; i++) {
        console.log(items[i]);
      }
  1. this for loop instatiates a variable i: let i = 0
  2. then sets a boolean statement that says, as long as i is less than items.length increase i (which is the i++ part)
  3. everytime you enter the loop you recieve a console.log with the item at index i
  4. once i is increased to equal or be greater than items.length the loop will end
