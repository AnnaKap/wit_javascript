Welcome to level 4! The land of if statements

Within functions we can use if/else statement conditions to make conditional logic

Let's build a function called divide that takes two arguments a & b and returns its result when divided

  • open level4.js in vscode
  • declare a function called divide
  • pass in 2 arguments: a & b and return the result of dividing the two
        
          function subtract(a,b) {
            return a/b;
          }

          let five = divide(10,2);
          console.log("10 divided by two is: ", five);

What happens however, if a user inputs 0 into substract as variable b?

  • call your function divide and pass in 10 for a and set it to a new variable
  • pass in 10 for a, and pass in 0 for b
  • pass in 2 arguments: a & b and return the result of dividing the two
  • check your console log, what do you get?
        
          function divide(a,b) {
            return a/b;
          }

          let answer = divide(10,0);
          console.log("10 divided by 0 is: ", answer);
          
you should get this error: Uncaught ReferenceError: divide is not defined, because you cannot divide by 4

Using if statements and a nifty throw javascript function we can prevent this error from happening!

  • edit the divide function by creating an "if statement" within the function
  • this if statement will catch any function calls where b is zero
  • check for if b is ever zero by using the === operator within your if statement
  • look for syntax help in the code below
    •           
              // this is your if statement
              if (b===0) {
                ..do something
              }
  • write a throw statement to customize the error your user will see
  • look for syntax help in the code below
    •           
              throw "your error in quotes";
          
        
          
            function divide (a,b) {
              if(b===0){
                throw 'cannot divide by zero';
              }
              return a/b;
            }
  
            let answer = divide(10,0);
            console.log("10 divided by 0 is: ", answer);
