Lets work on the addColor method

  • to access and set the styles of an html element we can write element.style
  • so in our case our targeted element is this.box, that means we need to write this.box.style to access/set styles
  • more specifically we want to access the color style, so we will write this.box.style.backgroundColor
  • add this line to your add color method and set it equal to a string of a color (ie "blue")
  • let's intialize a new box class and pass it the element id box as the argument

Lets customize addColor so a user can choose a color

  • add an argument to add color and set this.box.style.backgroundColor to equal this value
  • now call your method addColor and pass in a string of a different color (maybe "yellow"?)
      
        class Box {
          constructor(elementId){
            this.box = document.getElementById(elementId)
          }
        
          addColor(color) {
            this.box.style.backgroundColor = color;
          }
        }
        
        let myBox = new Box('box');
        myBox.addColor('yellow');

Ok now that we've done the square lets do the same for the circle, but this time you got it!

  • declare a class called circle
  • pass an elementId as an argument to the consturctor and use it to target the element and set it as the value this.cirlce
  • our circle is looking a bit square so lets round it up
  • within the constructor apply a style of borderRadius and set it equal to '50%'
  • now lets add a similar addColor method as box has
  • instatiate the circle class and add a color to it by calling the add color method
      
        class Circle {
          constructor(elementId){
            this.circle = document.getElementById(elementId);
            this.circle.style.borderRadius = '50%';
          }
        
          addColor(color) {
            this.circle.style.backgroundColor = color;
          }
        }
        
        let myCircle = new Circle('circle');
        myCircle.addColor('orange');
